Jest:

This project is concerned with detailing Unit Testing in JavaScript using Jest. This is suitable for large projects.

Basic testing practices and technologies are vital to learn.

Jest is a library for testing JavaScript code. It is an open source project maintained by Facebook. It is especially useful when testing React applications.

Remember to run:
npm init -y
npm I -D jest

Make sure to create a .test.js file containing the testing function.

Then go 'npm test' to carry out the test.

For working with Asynchronous data install 'axios'
npm i axios
