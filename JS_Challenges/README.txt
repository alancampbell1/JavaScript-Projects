This project focuses on JavaScript challenges, specifically focused on those asked in web development interviews.

Topics covered include:
- Reversing a String
- Checking if a word is a Palindrome
- Reversing an int
- Capitalising the first letter in each word in a sentence
- Finding the most common letter in a String
- The FizzBuzz challenge
- Longest word in an input
- Array Chunking
- Flatten Array
- Anagrams
- Letter Changes
- Add All
- Sum Primes
- Seek & Destroy
- Sort by Height
- Missing Letters
- Summing even and odd numbers
